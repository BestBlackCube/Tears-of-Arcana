# Tears-of-Arcana
## 게임 소개
오랜 시간 연구자로 일생을 받쳐온 주인공은 기밀리 진행되는 대형 프로젝트의 임원 중 한명으로써  
프로젝트 진행중, 팀원의 실수로 인해 연구소에서 대폭발이 일어나며, 다수의 연구원들과 연구자료가  
소멸 하였다.

해당 프로젝트의 유일한 생존자로써 살아남은 주인공은 대폭발의 모든 원인을 뒤집어 씌워지며,  
많은 시민과 같은 연구자들에게 비판과 불명예를 떠안으며 생활 해왔다.

점차 PTSD로 환각과 악몽을 꾸며, 이상과 현실을 구분 하지 못할정도로 내몰려져 있었는데…  
어느날 악몽에 등장하는 악마 “니바스” 그는 주인공을 자신의 게임에 초대하여 주인공에게 시련을 
줍니다.

[slay the Sprire](https://store.steampowered.com/app/646570/Slay_the_Spire/)을 벤치마킹하여  만든 턴제전략 카드 게임입니다.  
여러가지 속성 카드를 사용하여 공격, 방어, 회복, 저주등 전략을 구상하여 다양한 적, 맵의 시련을 돌파하며  
현실로 돌아가는 게임입니다.

## 프로젝트 소개
저희 기업 연계 프로젝트 10조의 주제는 "게임"으로 선정하여 각자 팀원들의 역량을 파악하며,  
플랫포머 게임 개발도 좋지만, 인디 게임을 데모 버전까지 개발해보자는 의지로 2D 턴제 카드 게임을  
개발 하였습니다. 

## 프로젝트 기획
인디 게임 턴제전략 카드 게임을 개발에 여러가지의 레퍼런스를 분석하여 초기 기획안을 작성하여  
기본적인 UI, 레벨 디자인을 만들고, 사용할 에셋, 소스, 오디오등 제작하거나 소스를 가져왔습니다.  

### 팀원 역할
- ### 기획: 레퍼런스 분석 및 레벨 디자인, 게임 시스템 설계
    - 레퍼런스를 분석하여 인디게임의 사용할 카드게임 구도와 GUI, 카드 상호작용의 시스템을  
    기획하고, 게임 스토리를 작성하여 인게임에 스토리를 풀어 나가는 스토리텔링도 병행한다.
- ### 게임 로직: 게임 시스템 구현, 클라이언트 로직 개발
    - 기획을 통해 정해진 게임 시스템과 다양한 카드게임의 레퍼런스를 통해 사용할 시스템을  
    벤치마킹하여 전체적인 로직을 개발하고, 게임플레이에 지장이 없도록 테스트를 거쳐  
    버그를 개선한다.
- ### 오디오: 오디오 소스 제작 및 편집
    - CC0 (Creative Commons Zero)를 사용하여 전반적인 오디오 소스를 확보하여 원본 소스는  
    그대로 사용하지만, 필요한 소스는 편집 프로그램을 사용하여 제작한다.
- ### GUI: 이미지, 에셋, UI 소스 제작 및 편집
    - 일러스트, 배경 같은 이미지는 스테이블 디퓨전 AI를 사용하여 이미지를 제작하고, 캐릭터  
    같은 경우 유니티 에셋 또는 itch.io 등을 사용하여 무료, 유로 버전의 소스를 사용한다.  
    또한 Aseprite를 사용하여 직접 제작을 한다.
